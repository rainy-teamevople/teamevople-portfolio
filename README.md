# Team Evople Portfolio

㈜팀에버플 리더십 포트폴리오 웹사이트입니다.

## 페이지

| 파일 | 설명 | URL |
|------|------|-----|
| `index.html` | 박한준 대표이사 포트폴리오 | `/` |
| `hwasun.html` | 박화선 리더 포트폴리오 | `/hwasun` |

## 기술 스택

- 순수 HTML/CSS/JavaScript (빌드 도구 없음)
- 다크/라이트 모드 토글 지원
- 반응형 디자인 (모바일 대응)
- Vercel 배포

## 로컬 실행

브라우저에서 `index.html` 또는 `hwasun.html`을 바로 열면 됩니다.
